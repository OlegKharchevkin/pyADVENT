===advaction===
+  названия действия
a  условие = результат

===advcave===
+  номер локации
s  короткое описание (необязательно; одна строка)
l  полное описание (если нет, то >; количство строк >= 1 ) 
t  условие = результат

===advclass===
счет(3 символа) сообщение(может состоять из нескольки строк)

===advevent===
e  условие = результат

===advmessa===
num сообщение(может состоять из нескольки строк)

===advobjec===
+  названия объекта
l(f если неподвижный)  растоложение объекта
i  описание для инвентаря
num описание состояния(> если нет) 

для счетчиков описание и растоложение необязательны
=  после идут сокровища (отмечаются * в инвентаре)

===advvocab===
d  полные синонимы направлений

s  слова, для которых существует тривиальный отпугивает
m  ответ

===Условия===
Стандартный вид
MARKER+arg1=arg2
obj - объект над которым производится действие

Маркеры инверсии
+  маркер отсутствия инверсии
-  маркер инверсии
Пример: h-ключ True если нет ключей

Маркеры
o obj == arg1 (если arg1 == "****", то сравнивается с "")
h arg1 в этой локации или в инвентаре (если arg1 пустой то на его место подставляется obj)
a arg1 в этой локации (если arg1 пустой то на его место подставляется obj)
t arg1 в инвентаре (если arg1 пустой то на его место подставляется obj)
% с вераятностью arg1
p состояние arg1 == arg2 (если arg1 пустой то на его место подставляется obj)
f arg1 неподвижный (если arg1 пустой то на его место подставляется obj)
+ всегда True
y выводится сообщение arg1, если ответ {'y', 'yes', 'д', 'да'}, то True
l игрок в локации arg1
> количество предметов в инвентаре > arg1
$ первый раз с вераятностью arg1, затем всега False
w аргументы записываются так: w+( arg arg arg arg ); obj == одному из аргументов

===Действия===
Cтандартный вид
MARKER.arg1=arg2

Маркеры
d добавить arg1 в текущую локацию (если arg1 пустой то на его место подставляется obj)
* удалить arg1 (переместить в локацию 999; если arg1 пустой то на его место подставляется obj)
c подобрать arg1 (если arg1 пустой то на его место подставляется obj)
m вывести сообщение arg1 (сообщения 994 - 999 могут вызываться движком для собственных нужд)
p изменить состояние arg1 на arg2 (если arg1 пустой то на его место подставляется obj)
l переместить игрока в локацию arg1
# специальные действия
t переместить arg1 в локацию arg2 (если arg1 пустой то на его место подставляется obj)
a увеличить состояние arg1 на 1 (если arg1 пустой то на его место подставляется obj) 
i показать предметы в локации 
n запустить проверку заново с obj = arg1
" в следующих строках без маркера - сообщение которое надо вывести

===Специальные действия===
действие зависит от arg1
1 закрыть игру
2 отобразить счет (score = !brought * 15 + !seen * 5)
3 отобразить счет и закрыть игру
4 короткие сообщения при повторном попадании в локацию
5 отобразить ивентарь
6 отобразить полное описание местности
7 добавить !light в ивентарь
8 убрать !light из ивентаря
9 приостоновить игру

===Специальные объекты===
!light буквально свет 
!brought количество принесенных сокровищ (добавляет 15 очков)
!seen количество увиденных сокровищ (добовляет 5 очков)
